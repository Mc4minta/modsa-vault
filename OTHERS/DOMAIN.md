## Thai Name

วิชาการ

## Purpose

## Example Topics

- Academic Calendar
- Course Information
- Curriculum
- Examination

- Graduation

# Domain Design Principles

A domain should:

- Represent one major functional area of Student Affairs.
- Be stable over time.
- Contain many related topics.
- Avoid storing individual FAQ documents directly.

Topics should:

- Focus on one subject.
- Contain actual knowledge.
- Link to related topics using Obsidian backlinks.
- Be reusable across multiple domains when appropriate.
